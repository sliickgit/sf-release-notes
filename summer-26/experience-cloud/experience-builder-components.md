# Experience Builder Components

## Overview

Experience Builder gets more flexible screen-flow and upload components.

## What changed

- Radio Button Group can lay out screen-flow choices horizontally or vertically.
- File Upload supports files up to 10 GB.
- More screen flow components now support styling overrides.
- Static resource images can be inserted directly from Display Text.

## Notes for Agents

- This section is useful when you are customizing the site-building experience rather than the runtime experience.
