# Field Service Mobile

## Overview

Field Service Mobile covers the Data Capture and Native GIS Maps changes that let mobile workers collect structured data and work with map data in the Field Service app.

The mobile app is where the operational details of these updates are most visible.

## What changed

- Data Capture flows now support simplified screen components for mobile workers.
- Repeaters let builders collect repeated records from a data source inside a flow screen.
- Screen style and layout can be customized to make mobile data entry easier to follow.
- These changes are available in the Field Service mobile app for Android and iOS.
- Native GIS Maps lets mobile workers create and update Salesforce records directly from the map, while keeping offline maps current with automatic updates.
- On Android, map downloads continue in the background so progress is preserved when the app is interrupted.

## Notes for Agents

- Use this page when the change is about mobile data collection or map-driven field work, not timesheets or scheduling.
- The linked help pages show the exact screen components, flow-building steps, and GIS map behavior behind the feature.
