# Nonprofits

## Overview

Nonprofits gets a broader Agentforce Nonprofit update that spans fundraising, program and case management, outcome management, grants management, and setup discovery.

## What changed

- Nonprofit Cloud is now Agentforce Nonprofit.
- Fundraising can pause gift validations during data migration, and it adds a newer recurrence engine plus a self-service Donor Support agent.
- Program and case management gains AI-driven insights, a Participant Summary agent, and a Participant Management agent in Slack.
- Outcome Management adds ApplicationForm fields so outcome activities can be tied back to the right application and indicator assignment.
- Salesforce Go centralizes setup and feature discovery for Fundraising, Program Management, and Case Management.
- Grants Management uses a modernized direct-to-provider API model for nonprofit tax status lookups.
- The Nonprofit Connect REST API adds new gift-validation flags for migration and data-loading scenarios.
