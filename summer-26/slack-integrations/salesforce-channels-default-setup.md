# Salesforce Channels Default Setup

## Overview

Salesforce channels are available by default in new orgs.

## Official Release Note

- [Enjoy Slack-Powered Collaboration in New Salesforce Orgs](https://help.salesforce.com/s/articleView?id=release-notes.rn_slack_default_on.htm&language=en_US)

## Related help

- [Access Salesforce Channels in the Slack Panel](https://help.salesforce.com/s/articleView?id=release-notes.rn_slack_panel.htm&language=en_US)

## What changed

- New orgs in Enterprise and Unlimited editions get Salesforce channels by default.
- The Slack panel on record pages gives a more streamlined experience.
- Existing orgs using the Slack Lightning web component can switch to the Slack panel.
- Chatter is turned off by default in new orgs created in Summer '26 and later, so Slack channels become the primary collaboration path in those orgs.

## Setup Notes

- Enable Slack Channels for Records from Setup.
- Select the objects to enable.
- Turn on Show Slack Button on Selected Objects when ready.

## Notes for Agents

- This is the main collaboration and page-integration update in the section.
- Use the related help when you need the exact Slack-panel behavior or want to confirm the default-on experience in new orgs.
