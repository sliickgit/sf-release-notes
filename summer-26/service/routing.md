## What changed

- Omni-Channel continues to automate work-item routing.
- Standard Omni-Channel is scheduled for retirement.
- You can control queue access for superiors in the role hierarchy.
- Work items can be scheduled to route at a specific date and time.
- Work can be routed based on the original request date.
- Service reps can be protected from calls before they are ready.
- The routing story is now split between retirement planning, queue access control, scheduled routing, and telephony readiness.

## Notes for Agents

- Routing remains a foundational service operation in the release notes.
- If a routing workflow depends on Standard Omni-Channel, start migration planning now.
- Use the related articles when you need the retirement, scheduling, or queue-control mechanics behind the routing summary.
