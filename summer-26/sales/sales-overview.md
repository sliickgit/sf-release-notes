# Sales (Agentforce Sales)

## Overview

Sales Cloud is now Agentforce Sales. The Summer '26 release notes continue to use Sales Cloud in some application and documentation references.

## Official Release Note

- [Sales](https://help.salesforce.com/s/articleView?id=release-notes.rn_sales.htm&language=en_US)

Sales is one of the major product areas in the Summer '26 release notes. It covers features and updates that help sales teams improve productivity and operations across the May through August 2026 update window.

The concrete Summer '26 story here is mostly about the broader Agentforce Sales rename and the routing of sales-team changes into the sales-specific release area.

## What changed

- The product name now appears as Agentforce Sales in the release intro, although some Salesforce applications and documentation still say Sales Cloud.
- The release notes direct readers here for the latest sales-team productivity and operations updates.
