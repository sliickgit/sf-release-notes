# Summer '26 Top 10 Features

This is a subjective shortlist of the Summer '26 features with the highest business value, the strongest revenue impact, and the biggest security or risk-reduction value.

## Highest Business Value

1. [Apex default user-mode and sharing changes](development/apex.md) - Apex now defaults more user-facing work into user mode and sharing-aware behavior, which reduces accidental overexposure and makes secure coding the baseline.
2. [Agentforce Contact Center](service/agentforce-contact-center.md) - Service is being reorganized around an AI-first contact center, so voice, email, and messaging can be handled in one modern engagement model.
3. [AI-Assisted Self-Service](experience-cloud/ai-assisted-self-service.md) - Experience Cloud sites gain orchestrated self-service components, which can deflect cases and let customers resolve more issues without an agent.
4. [Field Service Scheduling and Optimization](field-service/field-service-scheduling-and-optimization.md) - Dispatchers get better routing and scheduling controls, which improves technician utilization and reduces wasted travel.
5. [Data 360 Reports and Dashboards](analytics/data-360-reports-and-dashboards.md) - Data 360 reporting adds stronger trend, hierarchy, and currency reporting, which gives leaders better visibility into the customer data layer.
6. [Consumption Agent](revenue-management/consumption-agent.md) - Usage overages can now be turned into quote-ready monetary output, which removes manual conversion work and speeds monetization.
7. [Salesforce Mobile App updates](mobile/salesforce-mobile-app.md) - The mobile app gets faster sign-in, better home navigation, notifications, and transcription, which improves daily productivity for field and office users.
8. [Security Enhancements](security-identity-privacy/security-enhancements.md) - Email domain verification, DKIM, MFA, and login restrictions are being tightened, which lowers the chance of account compromise and deliverability issues.
9. [Salesforce Order Management](commerce/order-management.md) - Order routing and delivery estimation become more flexible and AI-assisted, which helps teams move orders through fulfillment with less manual intervention.
10. [Agentforce Voice SIP Routing](agentforce/agentforce-voice-sip-routing.md) - Voice agents can route calls over SIP instead of only PSTN, which gives telephony teams a more flexible and cost-effective deployment option.

## Highest Revenue Impact

1. [Consumption Agent](revenue-management/consumption-agent.md) - usage overages are converted into quote-ready financial amounts, which speeds monetization and removes manual calculation work.
2. [Agentforce Contact Center](service/agentforce-contact-center.md) - service teams get a more unified AI-first contact model, which can improve resolution speed and customer experience.
3. [AI-Assisted Self-Service](experience-cloud/ai-assisted-self-service.md) - self-service sites can resolve more requests without an agent, which lowers case volume and support cost.
4. [Salesforce Order Management](commerce/order-management.md) - routing rules and delivery estimates become more flexible, which helps orders move through fulfillment faster.
5. [Marketing Cloud Next](marketing/marketing-cloud-next.md) - campaign execution and personalization get more agent assistance, which can improve pipeline creation and campaign throughput.
6. [Salesforce Mobile App updates](mobile/salesforce-mobile-app.md) - faster login, better navigation, and meeting transcription help sellers and service users work more quickly in the field.
7. [B2B Commerce](commerce/b2b-commerce.md) - storefront, pricing, and search improvements make it easier for buyers to find and purchase products.
8. [B2C Commerce](commerce/b2c-commerce.md) - the monthly rollout structure keeps store changes moving predictably, which matters for sites that ship continuously.
9. [Field Service Scheduling and Optimization](field-service/field-service-scheduling-and-optimization.md) - better dispatch and routing control helps technicians spend more time on billable work and less time traveling.
10. [Partner Cloud](partner-cloud/enhanced-partner-connectivity.md) - improved partner connectivity and branded communications support partner-led selling and channel engagement.

## Highest Security and Risk Reduction

1. [Apex default user-mode and sharing changes](development/apex.md) - Apex now defaults more user-facing data access into user mode and sharing-aware behavior, which reduces accidental privilege exposure.
2. [Security Enhancements](security-identity-privacy/security-enhancements.md) - email verification, DKIM, MFA, and login restrictions are tightening, which lowers compromise risk and improves trust in outbound email.
3. [Identity and Access Management](security-identity-privacy/identity-and-access-management.md) - older auth patterns are being retired, which improves auditability and reduces dependence on weaker sign-in flows.
4. [Salesforce Connect Cross-Org Adapter](customization/salesforce-connect-cross-org-adapter.md) - named credentials make cross-org authentication easier to reason about and reduce brittle integration setup.
5. [Health Check](security-identity-privacy/health-check.md) - weekly signals and new checks make it easier to spot posture issues before they become incidents.
6. [Security Center](security-identity-privacy/security-center.md) - better alert metrics help security teams track posture changes and react earlier.
7. [Salesforce Shield](security-identity-privacy/salesforce-shield.md) - stronger scanning and transaction-security coverage improves protection for sensitive data and activity.
8. [Experience Cloud Security and Sharing](experience-cloud/security-and-sharing.md) - private pages, file scanning, and trust gates are tighter, which reduces site-abuse and content-risk exposure.
9. [Named Credentials](security-identity-privacy/named-credentials.md) - callback and auth generation based on My Domain makes external auth paths more consistent and less error-prone.
10. [Release Updates](release-updates.md) - enforced behavior changes and future update deadlines can break old assumptions, so this page is the compatibility warning system.
