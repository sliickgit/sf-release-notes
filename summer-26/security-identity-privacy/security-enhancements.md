# Security Enhancements

## Overview

Salesforce is tightening security controls over time.

## What changed

- Email domain verification is required.
- Additional controls start in June 2026.
- MFA is required for all employee-license users, including sandbox access.
- Salesforce admins need phishing-resistant MFA.
- Login IP address restrictions are part of the requirements.
- The change is mostly about forcing stronger identity proofing and stricter login hygiene before users can access Salesforce.
