# Flow and Process Release Updates

## Overview

This section points readers to separate release updates for Salesforce Flow.

The Flow release updates are the place to watch for behavior changes that affect Flow Builder, scheduled flows, and Flow administration rather than the broader Salesforce release-update catalog.

## What changed

- Flow Builder can now create and configure Agentforce agents directly from within a flow.
- Scheduled flows can use custom batch sizes to help avoid governor limits and improve performance.
- Email template references can be used across environments.
- Fault paths on the Flow Builder canvas can be collapsed for easier visual scanning.
- The Add Prompt Instructions element makes it easier to find resources.

## Notes for Agents

- Use the main Release Updates section to track enforcement and compatibility changes.
- Use this page when you need the Flow-specific release-update context behind a change that appears in Flow Builder or a scheduled flow.
