# Flow Builder

## Overview

Flow Builder receives the largest set of automation updates in Summer '26.

## Official Release Note

- [Automation](https://help.salesforce.com/s/articleView?id=release-notes.rn_automate_monthly.htm&language=en_US)

## What changed

- Create and configure Agentforce agents directly from Flow Builder.
- Improve performance for scheduled flows with custom batch sizes.
- Evolve screen flows using natural language.
- Personalize email messages with content variables from Email Content Builder.
- Notify users with toast messages.
- Add custom property editors to input parameters.
- Collapse fault paths to focus on the main path.
- Improve resource picking in the Add Prompt Instructions element.
- Support better filtered picklists and date operators in decision logic.
