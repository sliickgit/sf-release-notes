# June 2026

## Overview

Data 360 features and changes are released monthly, and the Summer '26 notes point most of the concrete detail to June 2026.

## What changed

- Setup with Agentforce is available as a pilot or beta service for certain editions.
- Queue sharing can optionally grant access to superiors in the role hierarchy.
- Organization-wide defaults are recalculated in parallel for faster updates.
- The All Restricted Employees group can extend access for Unified Employee License users.
- Custom fields can be used as join keys on standard entities for related lists on direct DMO relationships.
- Custom fields on standard entities can be included in sandbox mergeback deployments.
- Copy field enrichments can auto-enable the correct permissions for eligible users.
- Multi-dimensional aggregatable calculated insights objects can be used as copy field enrichment sources.
- Problem Records DLOs isolate ingestion failures for easier troubleshooting.
- Data Processing Engine can assign Data Model Object categories for downstream compatibility.
- Writeback operation types can be configured for Data Cloud objects.
- Data preview for Data Cloud Runtime helps validate logic and debug faster.
- Data Cloud usage and access changes affect perms, licensing, and billing.
- Connect REST API improvements are called out for developers.
