# MuleSoft for Flow: Integration

## Overview

MuleSoft for Flow adds centralized value mapping management.

## Official Release Note

- [Automation](https://help.salesforce.com/s/articleView?id=release-notes.rn_automate_monthly.htm&language=en_US)

## What changed

- Manage value translation lookups in one place.
- Standardize translations and improve flow stability when values are missing.
- Track updates across multiple integrations from a single interface.
- The new centralized management model reduces drift between different Flow integrations that reuse the same translated values.
