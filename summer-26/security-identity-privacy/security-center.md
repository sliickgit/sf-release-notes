# Security Center

## Overview

Security Center gains new alert metrics.

## Official Release Note

- [Security, Identity, and Privacy](https://help.salesforce.com/s/articleView?id=release-notes.rn_security.htm&language=en_US)

These metrics make it easier to see security posture changes and follow up on emerging issues.

## What changed

- New alert metrics improve security posture visibility.
- The release notes place this update in the Security Center section alongside other monitoring and hardening changes.
- The useful change is better alert-level visibility when you are tracking posture over time.

## Notes for Agents

- This is a small monitoring-oriented update.
- Use these metrics when you need a quick way to monitor the org's security signals without leaving the Security Center workflow.
- Treat this as a metrics improvement, not a broader Security Center redesign.
