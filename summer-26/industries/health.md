# Health

## Overview

Health gets faster setup, AI-assisted decision support, and improved self-service and pricing transparency.

## Official Release Note

- [Health](https://help.salesforce.com/s/articleView?id=release-notes.rn_health.htm)

## What changed

- Agentforce Health setup can be expedited with Salesforce Go.
- Agentforce can generate assessment suggestions.
- Specific page data can be extracted.
- Prior authorization and provider network status details can be summarized.
- Health Engagement supports outreach campaigns and member self-service.
- Digital Health Insurance supports plan comparisons.
- Digital Wallet improves pricing transparency for atheahealth API operations.

## Notes for Agents

- This cloud leans heavily on guided member and provider experiences.
- Agentforce is being used as a setup and summarization aid, not just an end-user chatbot.
