## What changed

- Setup with Agentforce adds an AI-powered setup assistant for administrative tasks in supported editions.
- Queue sharing can now optionally open access up the role hierarchy, and large sharing recalculations run in parallel for speed.
- The All Restricted Employees group now extends access for Unified Employee License users.
- Data Cloud relationship handling expands with custom join keys on standard entities and sandbox mergeback support for custom fields.
- Copy field enrichments can auto-enable the right permissions and can use multi-dimensional aggregatable calculated insights as a source.
- Problem Records DLOs isolate ingestion failures so teams can debug bad loads instead of hunting through the whole pipeline.
- Data Processing Engine can assign Data Model Object categories, configure writeback operation types, and preview data before activation.
- Data Cloud usage and access changes affect permissions, licensing, and billing behavior.
- Connect REST API updates are surfaced for developers who integrate against Data 360.
