# Tableau

## Overview

Tableau remains the broader analytics product family referenced by Salesforce.

This page is mostly a routing note that points agents toward the Tableau product-specific release notes when the Summer '26 release notes mention Tableau.

## What changed

- Tableau Cloud, Tableau Desktop, Tableau Prep, and Tableau Server each have their own release notes.
- Tableau visualizations can be embedded into Lightning pages and workflows.
- Tableau is positioned as the family for deeper data exploration.

## Notes for Agents

- Use the product-specific Tableau release notes when you need version-level detail.
- If you are looking for feature behavior in Summer '26, treat this page as the pointer and the other Tableau notes as the source of detail.
- Use it as the entry point when a Salesforce note mentions Tableau, but not as the place to look for version-specific behavior.
