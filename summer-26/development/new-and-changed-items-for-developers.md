## What changed

- This section is the item-level change log for Development, so it captures the exact names behind the broader developer notes.
- Apex changes here include the new user-mode and sharing defaults, trigger runtime behavior, the removal of WITH SECURITY_ENFORCED at newer compile levels, and the new Web Console.
- API changes here include Hosted MCP Servers, Context MCP Server, Named Query API, SOAP JWT, SOQL WHERE, and OpenAPI support for sObjects REST API.
- Lightning changes here include Live Preview, LWC versioning, Lightning Web Security updates, and component accessibility improvements.
- Tooling and packaging changes here include Agentforce DX, the Salesforce DX MCP Server, CLI and VS Code updates, sandbox-selection behavior, and push-upgrade expiration.
- Marketplace changes here include AgentExchange partner discovery, the marketplace navigation update, and connected-app security changes.
