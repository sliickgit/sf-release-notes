# Omnistudio Overview

## Overview

Omnistudio is Salesforce's low-code suite for building guided, industry-specific digital experiences.

It is the umbrella for the Omnistudio runtime and design tools used to create step-by-step customer and employee journeys in industry clouds.

## What changed

- It shortens the path from design to production for guided digital experiences.
- It supports structured, step-by-step interactions rather than free-form pages.
- It can combine Salesforce data with external data sources in the same guided flow.
- It is most relevant when teams are building industry-specific journeys that need low-code orchestration.
