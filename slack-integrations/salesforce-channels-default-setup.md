# Salesforce Channels Default Setup

## Overview

Salesforce channels are available by default in new orgs.

## What changed

- New orgs in Enterprise and Unlimited editions get Salesforce channels by default.
- The Slack panel on record pages gives a more streamlined experience.
- Existing orgs using the Slack Lightning web component can switch to the Slack panel.

## Setup Notes

- Enable Slack Channels for Records from Setup.
- Select the objects to enable.
- Turn on Show Slack Button on Selected Objects when ready.
