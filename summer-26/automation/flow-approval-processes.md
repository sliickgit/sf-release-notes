# Flow Approval Processes

## Overview

Flow Approval Processes get stronger approval and dependency visibility.

## Official Release Note

- [Automation](https://help.salesforce.com/s/articleView?id=release-notes.rn_automate_monthly.htm&language=en_US)

## What changed

- Approval steps can require unanimous approval.
- Approval designers can view flow dependencies for any approval process they create.
- Unanimous approval changes the decision rule for sensitive approvals, which can tighten governance.
- Dependency visibility makes it easier to understand what a change in one approval process might affect elsewhere.

## Notes for Agents

- This is a targeted but useful governance improvement.
- Treat this as an approval-control update rather than a broad Flow redesign.
