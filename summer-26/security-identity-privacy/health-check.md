# Health Check

## Overview

Health Check adds new signals and default weekly notifications.

The feature is aimed at making org-health monitoring easier to act on.

## What changed

- New signals help keep orgs secure.
- Admins receive weekly notifications by default.
- The new signals help surface security posture changes without requiring constant manual review.

## Notes for Agents

- This is mostly an admin-visibility improvement.
- Use it as the weekly check-in point for security posture rather than as a one-time setup item.
