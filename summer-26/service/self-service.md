# Self Service

## Overview

Self Service uses Agentforce Orchestrator and Concierge to route Experience Cloud conversations and improve user interaction.

The Experience Cloud self-service updates are now centered on AI-assisted conversation routing and guided support surfaces.

## What changed

- Agentforce Orchestrator can route conversations to the right AI agents and workflows.
- The prompt bar, suggestion themes, concierge sidebar, chat history, and dynamic content tiles are part of the self-service experience.
- These components are available for authenticated and guest users on Experience Cloud sites.
- Users can also get personalized recommendations and guided interactions from the same AI-assisted suite.

## Notes for Agents

- This is the AI-facing entry point for Experience Cloud self-service in Service.
- Use this page when you need to understand how the self-service experience is orchestrated across multiple Experience Cloud components.
