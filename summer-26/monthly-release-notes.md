# Monthly Release Notes

## Overview

Some Salesforce products release features and enhancements more frequently than the standard seasonal cadence. This section exists so you can track those monthly notes in one place.

## How to use this section

- Check it whenever a product says its changes are released monthly.
- Use it as the index for sections that change between major releases.
- Treat the monthly notes as the source of truth for incremental product updates.
- Expect this section to point to product-specific notes such as service, commerce, or data updates that are published outside the main seasonal cadence.

## Notes for Agents

- This section is a navigation aid as much as a content area.
- When a product area is updated monthly, its deeper details usually live in a dedicated subsection under that product.
- If a seasonal page looks sparse, check here for a more recent monthly follow-up.
