# Aura and LWR Sites

## Overview

Aura and LWR sites get improvements for self-service, page access, malware scanning, data tables, and secure vendor connections.

## What changed

- AI-assisted Self-Service components deliver personalized experiences.
- Private pages can stay unavailable until visitors finish setting up or resetting a password.
- Salesforce Files malware scanning is generally available.
- Data Table lookup columns in screen flows can show record names instead of Salesforce IDs.
- Record names can link to the related record in a new browser tab.
- A secure vendor connection can pull records from a vendor Aura site into Salesforce.
- Chatter-dependent features can be enabled for new orgs.
