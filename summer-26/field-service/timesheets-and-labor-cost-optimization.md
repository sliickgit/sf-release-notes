## What changed

- Field technicians can edit and delete timesheets with Agentforce natural-language commands.
- Crew leads can create and manage crew timesheets directly in the mobile app.
- Users can generate multiple timesheets for a date range.
- Enhanced summary views show regular time, time and a half, and double-time details before approval.
- Automatic idle time entries fill gaps between work periods.
- Rejected timesheets can be corrected and resubmitted.
- Mobile users can refresh timesheet data with a pull-down gesture.
- The feature family covers both mobile workflow handling and back-office review, so it matters to technicians and approvers.
