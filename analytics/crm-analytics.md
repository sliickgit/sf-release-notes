# CRM Analytics

## Overview

CRM Analytics includes dashboard, lens, analytics app, and data integration updates.

## What changed

- Optimized append, upsert, and delete actions improve write performance when outputting to existing Data Lake Objects.
- Output to Azure Data Lake is generally available through the Azure Data Lake output connector.
- Recipe results can be written to an output connection for downstream use.
