# Mobile Publisher

## Overview

Mobile Publisher gets project-management improvements.

Mobile Publisher is the mobile publishing workflow for managing app projects, so this page is mainly about how teams plan and move mobile changes through the release process.

## What changed

- Project-management workflows in Mobile Publisher are improved.
- The release intro groups this with other mobile updates such as Agentforce Voice integration and Salesforce mobile app meeting transcriptions.
- Mobile Publisher now covers both project lifecycle management and permission-justification changes for app store approvals.
