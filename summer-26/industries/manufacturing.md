# Manufacturing

## Overview

Manufacturing setup is being streamlined with Salesforce Go and expanded analytics support.

## Official Release Note

- [Manufacturing](https://help.salesforce.com/s/articleView?id=release-notes.rn_manufacturing.htm)

## What changed

- Salesforce Go for Manufacturing now includes Tableau Next Analytics.
- The Distributor Management System is included in the guided setup experience.
- The goal is to reduce time to value through a more streamlined and guided configuration path.
- The practical effect is that manufacturing teams get faster setup plus analytics earlier in the deployment process.

## Notes for Agents

- This subsection is more about setup acceleration than new end-user functionality.
- Use this page when the change is about getting manufacturing capabilities configured faster.
