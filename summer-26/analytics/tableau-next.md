# Tableau Next

## Overview

Tableau Next is Salesforce's end-to-end, agentic analytics experience.

## What changed

- Tableau Next combines Tableau with Agentforce's context-aware AI and the Data Cloud semantic layer.
- Features and changes are released monthly.
- The section is organized by functional area rather than one monolithic feature list.

## Functional areas covered

- Setup and Workspaces
- Data
- Semantics
- Agentforce for Analytics
- Metrics
- Visualizations and Dashboards
- Development
- Apps and Learning
