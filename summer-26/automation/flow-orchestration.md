# Flow Orchestration

## Overview

Flow Orchestration is now available as a standard feature in applicable editions.

## Official Release Note

- [Automation](https://help.salesforce.com/s/articleView?id=release-notes.rn_automate_monthly.htm&language=en_US)

## What changed

- Flow Orchestration moved into standard availability for supported editions.
- Teams no longer need a separate feature-flag-style rollout for the core orchestration capability in those editions.

## Notes for Agents

- This is mostly an availability and licensing change.
- Treat this page as a deployment and entitlement signal rather than a functional redesign.
- If you are checking rollout readiness, the key question is whether the org has the edition that now includes orchestration by default.
- If rollout timing matters, this page tells you whether Flow Orchestration is now part of the standard entitlement set.
