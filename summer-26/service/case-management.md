# Case Management

## Overview

Case Management gets workflow, routing, and AI-assistance improvements.

## What changed

- Duplicate cases can be merged in Omni-Channel queues.
- An Agentforce action can identify the best available experts for a case.
- Problem statements can be captured in rich text.
- Generative AI can help draft case comments.

## Notes for Agents

- The main pattern here is faster case resolution with less manual triage.
