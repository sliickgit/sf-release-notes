## What changed

- Faster email-based login simplifies sign-in.
- A redesigned central home page makes key data easier to reach.
- One-tap push notifications help users act on business changes immediately.
- AI-powered transcription captures meeting notes and follow-up tasks automatically.
- The user-facing result is a more direct, productivity-oriented mobile shell for day-to-day work.
- The mobile app now combines navigation, notifications, transcription, and sign-in changes into one more configurable user experience.
