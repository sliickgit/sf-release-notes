# Media

## Overview

Media updates focus on campaign execution, AI-assisted sales productivity, and data management.

## What changed

- Television, radio, and digital placements can be managed in a single interface.
- AI agents can create ad proposals and pitch decks.
- Bulk targeting and frequency caps help standardize campaign reach.
- Promotions are configurable for B2B and B2C markets.
- Built-in DPE definitions and dynamic named credentials improve data management and observability.

## Notes for Agents

- This is a broad operational update for media teams rather than a single feature release.
