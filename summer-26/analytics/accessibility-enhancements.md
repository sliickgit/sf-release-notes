# Accessibility Enhancements

## Overview

Analytics includes smaller accessibility improvements that affect how dashboards and reports are consumed.

The release-note details here are mostly about how screen readers, focus states, and zoom behavior interact with charts and tables.

## What changed

- Screen reader and ARIA updates improve how dashboards, filters, and tables are announced.
- Color contrast and focus visibility improvements help low-vision users navigate more reliably.
- Zoom behavior is improved so chart status text does not truncate at high magnification.

## Notes for Agents

- This section matters for usability and compliance reviews.
- Treat this as the analytics accessibility checklist, not as a new feature announcement.
- Use it when validating whether chart and dashboard behavior works well with assistive technology.
